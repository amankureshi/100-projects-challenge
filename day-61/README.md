# ⏱️ Project 60: Word & Letter Counter using React.js & Tailwind CSS

🔗 **Live Demo:** [ https://word-and-letter-counter.onrender.com/]

## 📄 Description

A simple yet powerful **Word & Letter Counter App** built with **React.js** and **Tailwind CSS** that allows users to count words, letters, and characters in real time as they type.

## 🔧 Technologies Used

- React.js
- Tailwind CSS
- useState Hook

## 🌟 Features

- Real-time word and letter counting
- Clear text functionality
- Responsive and clean UI with Tailwind CSS
- Handles multiple spaces and punctuation gracefully

## 🚀 Challenges Faced

- Handling extra spaces and special characters correctly
- Maintaining real-time updates while typing
- Ensuring responsive layout using Tailwind utilities

## 🎯 Key Learnings

- String manipulation and counting logic in React
- State handling for dynamic text updates
- Building minimal yet useful text utility tools with React & Tailwind CSS
